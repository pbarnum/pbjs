Customizing Boxes
The create() method will accept one argument of type Object. Listed below are all the options available to you...

int parentSets the parent Box for the new Box.
bool isMovableAllows the Box to move within the window. The title bar is used to move the Box.
bool isScalableAllows the Box to scale. A div element will be appended to the Box for its control.
object buttonsReads a key:value object where the key represents the button text and the value its function.
int heightThe height of the Box.
int widthThe width of the Box.
string titleThe title of the Box.
mixed bodyThe body can be represented as a function, or string.
function beforeCreateCallback function returning one argument: array existingBoxes.
function afterCreateCallback function returning one argument: object newBox.
function beforeCloseCallback function returning one argument: object currentBox.
